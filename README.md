# All-Pairwise-Computation---MPI
Parallel-Computing-and-Distributed-Systems USYD

All Pairwise Computation problem the computation is performed for every pair of items in
a dataset. In this assignment, we focus on computing the dot product for all pairs of column
vectors in a given matrix. Each column represents a sequence of random values assigned by
providing a particular seed, 1558 in our case.
